# My Home Cares: Keyword-to-Page Map

This maps the searches you want to rank for to the exact page that should answer them. Each page has ONE primary keyword (its main job) and a few secondary keywords it should also mention naturally. Google ranks pages, not sites, so every important search needs a clear "home" page.

How to read it: **Primary** = the main phrase the page targets (put it in the title and H1). **Secondary** = related phrases to work into the copy naturally. Don't stuff, write for people first.

## The honest truth about timing

On-page targeting tells Google what each page is about, that part is now solid. But ranking for competitive local terms also depends on:

1. **Google Business Profile** (the single biggest local factor), claim, verify, and fill it out.
2. **Reviews**, volume and recency strongly influence local ranking.
3. **Time and links**, a new domain earns trust over weeks to months; local citations and backlinks speed it up.

So think of today's work as making sure that when Google is ready to rank you, it ranks you for the right things. Pair it with the Local SEO checklist (GBP + reviews + directories) and the two compound over the next few months.
